# 开发环境搭建说明

## 📦 项目包说明

现在有两个不同用途的压缩包：

### 1. 🚀 生产部署包
- **文件名**: `my-resume-builder-deploy.tar.gz` (4.3MB)
- **用途**: 宝塔等服务器部署
- **包含**: 已构建的生产文件 + 源码
- **特点**: 包含 `.next` 构建文件，可直接运行

### 2. 💻 开发源码包  
- **文件名**: `my-resume-builder-source.tar.gz` (3.3MB)
- **用途**: 本地开发环境搭建
- **包含**: 纯源码 + 配置文件
- **特点**: 不包含 `node_modules` 和 `.next`，需要重新安装

## 🛠️ 开发环境搭建步骤

### 前置要求
- Node.js >= 18.0.0
- npm 或 yarn 包管理器

### 1. 解压源码包
```bash
# 解压到开发目录
tar -xzf my-resume-builder-source.tar.gz
cd my-resume-builder/
```

### 2. 安装依赖
```bash
# 安装所有依赖（包括开发依赖）
npm install

# 或使用 yarn
yarn install
```

### 3. 环境配置
```bash
# 复制环境变量文件（如果需要）
cp .env.local.example .env.local

# 编辑环境变量
nano .env.local
```

### 4. 启动开发服务器
```bash
# 启动开发模式
npm run dev

# 或使用 yarn
yarn dev
```

### 5. 访问应用
打开浏览器访问: `http://localhost:3000`

## 📋 开发命令说明

```bash
# 开发模式（热重载）
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start

# 代码检查
npm run lint

# 类型检查
npm run type-check
```

## 🚨 注意事项

1. **首次安装**: 需要运行 `npm install` 安装依赖
2. **环境变量**: 根据需要配置 `.env.local`
3. **端口冲突**: 默认 3000 端口，可在启动时指定其他端口
4. **内存要求**: 开发模式需要更多内存（建议 >= 2GB）

## 🔄 从部署包恢复开发环境

如果只有部署包，也可以恢复开发环境：

```bash
# 解压部署包
tar -xzf my-resume-builder-deploy.tar.gz

# 删除构建文件
rm -rf .next

# 安装开发依赖
npm install

# 启动开发模式
npm run dev
```
