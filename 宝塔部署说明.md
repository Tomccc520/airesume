# 简历编辑器 - 宝塔部署说明

## 项目简介

这是一个基于 Next.js 14 的在线简历编辑器，支持多种简历模板，具备AI智能优化功能。

## 部署方式选择

### 🚀 方式一：静态部署（推荐）
**适用场景**: 不需要AI功能，只需要基本的简历编辑功能
- **文件名**: `resume-static-deploy.tar.gz`
- **大小**: 约 3.6MB
- **优点**: 部署简单，无需Node.js环境，直接上传即可访问
- **缺点**: 不支持AI智能优化功能

### 🔧 方式二：Node.js部署（完整功能）
**适用场景**: 需要完整功能，包括AI智能优化
- **文件名**: `my-resume-builder-deploy.tar.gz`
- **大小**: 约 7.7MB
- **优点**: 支持所有功能，包括AI智能优化
- **缺点**: 需要Node.js环境和更多配置

## 静态部署步骤（推荐）

### 1. 下载部署包
下载 `resume-static-deploy.tar.gz` 文件

### 2. 上传到宝塔
1. 登录宝塔面板
2. 进入【文件】管理
3. 找到网站根目录（如：`/www/wwwroot/你的域名/`）
4. 上传 `resume-static-deploy.tar.gz` 文件
5. 解压文件：右键选择"解压"

### 3. 访问网站
直接通过域名访问即可，无需其他配置！

## Node.js 完整部署步骤

### 环境要求
- Node.js >= 18.0.0
- 宝塔面板已安装 Node.js 管理器

### 1. 上传部署包
1. 下载 `my-resume-builder-deploy.tar.gz`
2. 上传到宝塔面板的网站目录
3. 解压文件

### 2. 安装依赖
```bash
npm install --production
```

### 3. 环境配置
复制环境变量模板：
```bash
cp .env.example .env.local
```

编辑 `.env.local` 文件：
```env
NODE_ENV=production
PORT=3000

# AI功能配置（可选）
OPENAI_API_KEY=your_api_key_here
OPENAI_BASE_URL=https://api.openai.com/v1
```

### 4. 启动应用
```bash
npm start
```

### 5. 反向代理设置
在宝塔面板中设置反向代理：
- 目标URL: `http://127.0.0.1:3000`
- 发送域名: `$host`

#### 7. SSL 证书配置
在宝塔面板中为域名配置 SSL 证书，确保 HTTPS 访问。

### 📋 注意事项

1. **端口配置**: 默认运行在 3000 端口，确保端口未被占用
2. **内存要求**: 建议服务器内存 >= 1GB
3. **文件权限**: 确保 Node.js 进程有读写权限
4. **防火墙**: 开放 3000 端口或配置反向代理

### 🔧 常见问题

#### Q: 构建文件太大怎么办？
A: 已经排除了 `.next/cache` 目录（237MB），部署包只有 4.3MB

#### Q: 如何更新项目？
A: 重新上传新的部署包，解压后重启 Node.js 应用

#### Q: AI 功能不工作？
A: 检查 `.env.local` 中的 `OPENAI_API_KEY` 配置

### 📊 性能优化建议

1. **开启 Gzip 压缩**: 在宝塔面板中开启网站 Gzip 压缩
2. **静态文件缓存**: 配置 `.next/static/` 目录的长期缓存
3. **CDN 加速**: 可选择使用 CDN 加速静态资源
