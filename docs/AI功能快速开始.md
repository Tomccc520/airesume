# AI功能快速开始指南

## 🚀 5分钟快速上手

### 第一步：配置AI服务（首次使用）

1. 点击编辑器右上角的 **"AI配置"** 按钮
2. 选择 **"免费模型"** 选项（无需API密钥，开箱即用）
3. 点击 **"保存配置"**

> 💡 **提示**: 免费模型已预配置好，无需任何额外设置！

### 第二步：使用AI优化简历

#### 方式1: 单模块优化（推荐新手）

1. 在编辑器中找到要优化的模块（如"工作经历"）
2. 点击模块右上角的 **"✨ AI优化"** 按钮
3. 在弹出的AI助手中：
   - 查看当前内容
   - 点击快速建议按钮（如"量化工作成果"）
   - 或输入自定义需求（如"突出技术深度"）
4. 等待5-10秒，AI会生成5个不同风格的建议
5. 浏览建议，点击 **"应用"** 按钮采用喜欢的版本

#### 方式2: 分步生成（推荐新用户）

1. 点击编辑器顶部的 **"AI分步生成"** 按钮
2. 填写基本信息：
   - 姓名
   - 目标职位
   - 所在行业
   - 经验水平
3. 选择生成模式：
   - **逐步模式**: 每个模块单独生成，可以确认和修改
   - **快速模式**: 一次性生成所有模块
4. AI会按顺序生成：个人简介 → 工作经历 → 教育背景 → 技能 → 项目经历
5. 每个模块生成后，可以选择：
   - ✅ 确认并继续
   - 🔄 重新生成
   - ⏭️ 跳过此模块
6. 完成后在最终审核页面选择要应用的内容

#### 方式3: JD匹配优化（推荐求职者）

1. 点击编辑器顶部的 **"JD匹配"** 按钮
2. 粘贴目标职位的职位描述（JD）
3. 点击 **"开始分析"**
4. 查看匹配结果：
   - 匹配分数
   - 已匹配的关键词
   - 缺失的关键词
   - 优化建议
5. 点击建议卡片的 **"应用"** 按钮，自动优化简历

3. 粘贴JD并分析
4. 查看缺失的关键词
5. 逐个应用优化建议
6. 再次分析，确保匹配度达到80%+

**匹配度目标**:
- 🟢 90-100分：优秀，简历与JD高度匹配
- 🔵 75-89分：良好，有较大面试机会
- 🟡 60-74分：中等，建议继续优化
- 🔴 <60分：较低，需要大幅调整

## 🎯 AI建议选择技巧

### 如何选择最佳建议？

AI会生成5个不同风格的建议，每个都有特点：

1. **专业型** - 正式、规范，适合传统行业
2. **成果导向型** - 突出数据和成果，适合业务岗位
3. **技术专家型** - 强调技术深度，适合技术岗位
4. **管理型** - 突出领导力，适合管理岗位
5. **创新型** - 强调创新能力，适合创业公司

**选择建议**:
- 查看质量分数（系统会自动评分）
- 阅读每个建议的完整内容
- 选择最符合目标职位要求的风格
- 可以混合使用不同建议的优点

### 质量分数说明

每个建议都会显示质量评分：

- **90-100分** 🟢 优秀 - 可直接使用
- **75-89分** 🔵 良好 - 建议微调
- **60-74分** 🟡 中等 - 需要优化
- **<60分** 🔴 待改进 - 建议重新生成

**评分维度**:
- 清晰度：内容是否清晰易懂
- 具体性：是否包含具体细节和数据
- 专业性：是否使用专业术语
- 行动导向性：是否使用强有力的动作词
- 可量化性：是否包含量化成果

## ⚡ 高级技巧

### 技巧1: 迭代优化法

不要期望一次生成完美内容，采用迭代方式：

```
第1轮：生成基础内容
  ↓
第2轮：添加量化数据
  ↓
第3轮：提升专业性
  ↓
第4轮：突出核心亮点
  ↓
完成！
```

### 技巧2: 组合使用不同建议

从不同建议中提取优点：
- 从建议1中取量化数据
- 从建议2中取技术描述
- 从建议3中取表达方式
- 手动组合成最佳版本

### 技巧3: 自定义提示词模板

保存常用的提示词，提高效率：

**个人简介优化模板**:
```
优化个人简介，要求：
1. 突出[核心技能]
2. 体现[年]年经验
3. 强调[特长领域]
4. 控制在100字以内
```

**工作经历优化模板**:
```
优化工作经历，要求：
1. 使用STAR法则
2. 添加具体的量化数据
3. 突出[技术栈]
4. 强调团队协作和领导力
```

### 技巧4: 批量操作

对于多个相似的模块（如多段工作经历），可以：
1. 先优化第一个
2. 总结优化模式
3. 对其他模块应用相同的优化策略
4. 确保风格一致

## 🔍 常见问题

### Q1: AI生成的内容不准确怎么办？

**A**: AI生成的是建议模板，需要根据实际情况调整：
- 修改具体的数字和数据
- 调整技术栈和工具名称
- 补充个人特色和独特经历
- 删除不适用的内容

### Q2: 生成速度慢怎么办？

**A**: 
- 免费模型有一定延迟（5-15秒），请耐心等待
- 避免频繁重新生成，先看完所有建议再决定
- 可以配置付费API获得更快速度

### Q3: 如何避免简历千篇一律？

**A**:
- AI建议只是基础，一定要加入个人特色
- 使用真实的项目经历和数据
- 突出独特的技能和成就
- 手动调整表达方式

### Q4: 生成的建议太长或太短？

**A**:
- 使用"简化"意图：输入"简化内容，控制在XX字"
- 使用"扩展"意图：输入"扩展内容，添加更多细节"
- 手动编辑调整长度

### Q5: 如何确保简历真实性？

**A**:
- AI只是辅助工具，不要完全照搬
- 所有数据必须基于真实经历
- 技能和项目必须是真实掌握的
- 面试时要能详细解释简历内容
