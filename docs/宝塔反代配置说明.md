# 宝塔面板反向代理配置说明

## 部署方案

### 1. 静态文件部署
1. 将 `out` 文件夹的内容直接上传到网站根目录
2. 确保网站可以正常访问

### 2. AI功能启用（反向代理配置）

#### 方案一：使用外部AI服务（推荐）
在宝塔面板 → 网站 → 你的网站 → 反向代理，添加以下配置：

**代理名称：** AI API代理
**目标URL：** `https://api.siliconflow.cn`
**发送域名：** `api.siliconflow.cn`
**代理目录：** `/api/ai`

**高级配置：**
```nginx
# 添加必要的请求头
proxy_set_header Authorization $http_authorization;
proxy_set_header Content-Type $http_content_type;
proxy_set_header Accept $http_accept;

# 处理预检请求
if ($request_method = 'OPTIONS') {
    add_header 'Access-Control-Allow-Origin' '*';
    add_header 'Access-Control-Allow-Methods' 'GET, POST, OPTIONS';
    add_header 'Access-Control-Allow-Headers' 'DNT,User-Agent,X-Requested-With,If-Modified-Since,Cache-Control,Content-Type,Range,Authorization';
    add_header 'Access-Control-Max-Age' 1728000;
    add_header 'Content-Type' 'text/plain; charset=utf-8';
    add_header 'Content-Length' 0;
    return 204;
}

# 添加CORS头
add_header 'Access-Control-Allow-Origin' '*' always;
add_header 'Access-Control-Allow-Methods' 'GET, POST, OPTIONS' always;
add_header 'Access-Control-Allow-Headers' 'DNT,User-Agent,X-Requested-With,If-Modified-Since,Cache-Control,Content-Type,Range,Authorization' always;
```

#### 方案二：使用本地AI网关
如果你想要更多控制，可以部署本地AI网关：

1. 在服务器上创建AI网关目录：
```bash
mkdir /www/ai-gateway
cd /www/ai-gateway
```

2. 创建 `package.json`：
```json
{
  "name": "resume-ai-gateway",
  "version": "1.0.0",
  "description": "AI Gateway for Resume Builder",
  "main": "server.js",
  "scripts": {
    "start": "node server.js",
    "pm2:start": "pm2 start server.js --name resume-ai-gateway",
    "pm2:stop": "pm2 stop resume-ai-gateway",
    "pm2:restart": "pm2 restart resume-ai-gateway"
  },
  "dependencies": {
    "express": "^4.18.2",
    "cors": "^2.8.5"
  }
}
```

3. 创建 `server.js`：
```javascript
const express = require('express');
const cors = require('cors');
const app = express();
const port = 3001;

app.use(cors());
app.use(express.json());

// AI API代理
app.post('/api/ai', async (req, res) => {
  try {
    const response = await fetch('https://api.siliconflow.cn/v1/chat/completions', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': req.headers.authorization || ''
      },
      body: JSON.stringify(req.body)
    });
    
    const data = await response.json();
    res.json(data);
  } catch (error) {
    console.error('AI API Error:', error);
    res.status(500).json({ error: 'AI服务暂时不可用' });
  }
});

// 健康检查
app.get('/health', (req, res) => {
  res.json({ status: 'ok', timestamp: new Date().toISOString() });
});

app.listen(port, () => {
  console.log(`AI Gateway running on port ${port}`);
});
```

4. 安装依赖并启动：
```bash
npm install
pm2 start server.js --name resume-ai-gateway
```

5. 在宝塔面板添加反向代理：
**代理名称：** AI Gateway
**目标URL：** `http://127.0.0.1:3001`
**发送域名：** `$host`
**代理目录：** `/api/ai`

## 环境变量配置

如果使用AI功能，需要在前端设置API密钥。可以通过以下方式：

1. 在网站根目录创建 `config.js`：
```javascript
window.AI_CONFIG = {
  apiKey: 'your-api-key-here',
  baseUrl: '/api/ai'
};
```

2. 在 `index.html` 中引入：
```html
<script src="/config.js"></script>
```

## 常见问题

### Q: AI功能不工作怎么办？
A: 检查以下几点：
- 反向代理配置是否正确
- API密钥是否有效
- 浏览器控制台是否有错误信息

### Q: 如何更换AI服务提供商？
A: 修改反向代理的目标URL，或者更新AI网关中的API地址

### Q: 性能监控功能需要配置吗？
A: 性能监控是前端功能，不需要额外配置，会自动工作
