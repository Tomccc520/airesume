## 🐛 常见问题排查

### 问题1: 导出的PDF文字模糊
**原因**: DPI设置过低或PDF被压缩
**解决**: 已修复，现在使用300 DPI和无压缩设置

### 问题2: 导出时样式丢失
**原因**: CSS变量未解析或Flex布局问题
**解决**: 已修复，增强了样式预处理

### 问题3: 导出失败或卡住
**可能原因**:
- 字体加载超时
- 内容过于复杂
- 浏览器内存不足

**排查步骤**:
1. 打开浏览器控制台查看错误
2. 检查网络连接（字体加载）
3. 尝试减少内容后再导出
4. 刷新页面重试

### 问题4: 多页内容导出不完整
**原因**: 分页算法问题
**解决**: 已修复，实现了智能分页

### 问题5: 导出文件过大
**原因**: 图片质量设置过高
**解决**: JPG格式已优化为0.95质量，平衡质量和大小
