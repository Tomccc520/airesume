## 🎨 新增服务

### 1. AI提示词增强服务
**文件**: `src/services/aiPromptEnhancer.ts`

```typescript
// 自动识别意图并增强提示词
const enhancedPrompt = aiPromptEnhancer.enhancePrompt(
  '优化这段经历，突出团队协作',
  { sectionType, currentContent, targetPosition }
)
```

**功能**:
- ✅ 9种意图自动识别
- ✅ 上下文感知增强
- ✅ 专门模板生成

### 2. AI建议质量评估服务
**文件**: `src/services/aiQualityChecker.ts`

```typescript
// 评估建议质量
const score = aiQualityChecker.evaluateContent(content, 'experience')
console.log(score.overall) // 85分
console.log(score.issues) // 发现的问题
console.log(score.suggestions) // 改进建议
```

**功能**:
- ✅ 5维度评分
- ✅ 问题自动检测
- ✅ 改进建议生成

### 3. AI建议智能排序服务
**文件**: `src/services/aiSuggestionRanker.ts`

```typescript
// 智能排序建议
const ranked = aiSuggestionRanker.rankSuggestions(
  suggestions,
  'experience',
  { targetKeywords: ['React', 'TypeScript'] }
)
```

**功能**:
- ✅ 多维度排序
- ✅ 可配置权重
- ✅ 多样性保证
