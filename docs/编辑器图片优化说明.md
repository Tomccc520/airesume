# 编辑器图片优化说明

## 更新内容

已将 `src/components/ResumeEditor.tsx` 中的头像上传功能升级为使用优化的图片上传组件。

## 主要改进

### 之前（原生实现）

```tsx
// 使用原生 input 和 FileReader
<input
  type="file"
  accept="image/*"
  onChange={handleAvatarUpload}
  className="hidden"
  id="avatar-upload"
/>
<label htmlFor="avatar-upload">
  <Upload size={14} />
  选择图片
</label>
```

### 现在（优化实现）

```tsx
// 使用 OptimizedImageUpload 组件
<OptimizedImageUpload
  onUpload={handleAvatarUpload}
  currentImage={resumeData.personalInfo.avatar || styleConfig.avatar.url}
  maxWidth={400}
  maxHeight={400}
  quality={0.85}
  maxFileSize={2 * 1024 * 1024}  // 2MB
  showOptimizationInfo={true}
  showProgress={true}
/>
```

### 处理函数更新

```tsx
// 之前
const handleAvatarUpload = (event: React.ChangeEvent<HTMLInputElement>) => {
  const file = event.target.files?.[0]
  if (file) {
    const reader = new FileReader()
    reader.onload = (e) => {
      const result = e.target?.result as string
      // 处理结果...
    }
    reader.readAsDataURL(file)
  }
}

// 现在
const handleAvatarUpload = (dataUrl: string, blob: Blob) => {
  if (dataUrl) {
    // 更新头像URL到样式配置中
    updateStyleConfig({
      avatar: { ...styleConfig.avatar, url: dataUrl }
    })
    
    // 同时更新简历数据中的头像
    onUpdate({
      ...resumeData,
      personalInfo: {
        ...resumeData.personalInfo,
        avatar: dataUrl
      }
    })
    
    // 显示成功提示
    toast?.success('头像上传成功')
  }
}
```

## 注意事项

1. **文件大小**: 头像限制为 2MB，确保快速上传
2. **图片质量**: 85% 质量适合头像，保持清晰度
3. **尺寸限制**: 400x400 像素适合简历头像显示
4. **格式支持**: JPG、PNG、WebP 格式
