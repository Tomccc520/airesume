# 图片优化服务使用指南

## 📊 功能概述

图片优化服务提供完整的图片处理功能，包括：
- ✅ 图片压缩（减少文件大小）
- ✅ 尺寸调整（限制最大宽高）
- ✅ 格式转换（JPEG、PNG、WebP）
- ✅ 自动选择最优格式
- ✅ 保持图片质量

## 🎯 使用场景

1. **简历头像上传** - 压缩用户上传的头像，减少文件大小
2. **图片导出** - 优化导出到 PDF/Word 的图片质量
3. **性能优化** - 减少图片加载时间，提升页面性能

## 🚀 快速开始

### 1. 基础使用

```typescript
import { optimizeImage } from '@/services/imageOptimization';

// 优化单张图片
const handleFileUpload = async (file: File) => {
  const result = await optimizeImage(file, {
    maxWidth: 800,
    maxHeight: 800,
    quality: 0.8,
    format: 'auto', // 自动选择最优格式
  });

  console.log('原始大小:', result.originalSize);
  console.log('优化后:', result.optimizedSize);
  console.log('压缩率:', result.compressionRatio + '%');
  
  // 使用优化后的图片
  const imageUrl = result.dataUrl;
  const imageBlob = result.blob;
};
```

### 2. 在 React 组件中使用

```typescript
import { useImageOptimization } from '@/hooks/useImageOptimization';

function ImageUpload() {
  const { optimizeImage, optimizing, result } = useImageOptimization({
    maxWidth: 800,
    quality: 0.8,
    onOptimized: (result) => {
      console.log('优化完成:', result);
    }
  });

  const handleFileChange = async (e: React.ChangeEvent<HTMLInputElement>) => {
    const file = e.target.files?.[0];
    if (file) {
      await optimizeImage(file);
    }
  };

  return (
    <div>
      <input type="file" onChange={handleFileChange} />
      {optimizing && <p>优化中...</p>}
      {result && (
        <div>
          <img src={result.dataUrl} alt="预览" />
          <p>压缩率: {result.compressionRatio.toFixed(2)}%</p>
        </div>
      )}
    </div>
  );
}
```

## 📖 API 文档

### ImageOptimizationService

#### optimizeImage(file, options)

优化单张图片。

**参数：**
- `file: File` - 原始图片文件
- `options: ImageOptimizationOptions` - 优化选项
  - `maxWidth?: number` - 最大宽度（默认 800）
  - `maxHeight?: number` - 最大高度（默认 800）
  - `quality?: number` - 图片质量 0-1（默认 0.8）
  - `format?: 'jpeg' | 'png' | 'webp' | 'auto'` - 输出格式（默认 'auto'）
  - `maintainAspectRatio?: boolean` - 是否保持宽高比（默认 true）

**返回：**
```typescript
{
  blob: Blob,              // 优化后的 Blob 对象
  dataUrl: string,         // Data URL（可直接用于 img src）
  originalSize: number,    // 原始文件大小（字节）
  optimizedSize: number,   // 优化后大小（字节）
  compressionRatio: number,// 压缩率（百分比）
  width: number,           // 图片宽度
  height: number,          // 图片高度
  format: string           // 图片格式
}
```

#### optimizeImages(files, options)

批量优化图片。

**参数：**
- `files: File[]` - 图片文件数组
- `options: ImageOptimizationOptions` - 优化选项

**返回：**
- `Promise<ImageOptimizationResult[]>` - 优化结果数组

#### convertFormat(file, targetFormat, quality)

转换图片格式。

**参数：**
- `file: File` - 原始图片文件
- `targetFormat: 'jpeg' | 'png' | 'webp'` - 目标格式
- `quality?: number` - 质量（默认 0.9）

**返回：**
- `Promise<Blob>` - 转换后的 Blob

#### resizeImage(file, width, height)

调整图片尺寸。

**参数：**
- `file: File` - 原始图片文件
- `width: number` - 目标宽度
- `height: number` - 目标高度

**返回：**
- `Promise<Blob>` - 调整后的 Blob

#### supportsWebP()

检查浏览器是否支持 WebP 格式。

**返回：**
- `boolean` - 是否支持

#### formatFileSize(bytes)

格式化文件大小。

**参数：**
- `bytes: number` - 字节数

**返回：**
- `string` - 格式化后的字符串（如 "1.50 MB"）

### useImageOptimization Hook

React Hook，提供图片优化功能。

**参数：**
```typescript
{
  maxWidth?: number,
  maxHeight?: number,
  quality?: number,
  format?: 'jpeg' | 'png' | 'webp' | 'auto',
  autoOptimize?: boolean,
  onOptimized?: (result) => void,
  onError?: (error) => void
}
```

**返回：**
```typescript
{
  optimizeImage: (file: File) => Promise<ImageOptimizationResult | null>,
  optimizeImages: (files: File[]) => Promise<ImageOptimizationResult[]>,
  optimizing: boolean,
  progress: number,
  result: ImageOptimizationResult | null,
  error: Error | null,
  reset: () => void
}
```

## ⚙️ 配置建议

### 不同场景的推荐配置

| 场景 | maxWidth | maxHeight | quality | format |
|------|----------|-----------|---------|--------|
| 头像 | 400 | 400 | 0.85 | auto |
| 简历背景 | 1200 | 1600 | 0.8 | jpeg |
| 缩略图 | 200 | 200 | 0.75 | auto |
| 高质量图片 | 1920 | 1080 | 0.9 | auto |

### 质量参数说明

- `0.6-0.7` - 低质量，文件最小，适合缩略图
- `0.7-0.8` - 中等质量，平衡文件大小和质量
- `0.8-0.9` - 高质量，文件较大，适合重要图片
- `0.9-1.0` - 最高质量，文件最大，接近原图

## ⚠️ 注意事项

1. **文件大小限制**：建议限制上传文件大小（如 10MB）
2. **内存占用**：处理大图片时会占用较多内存
3. **格式支持**：仅支持 JPEG、PNG、WebP 格式
4. **透明度**：JPEG 不支持透明度，PNG 图片建议保持 PNG 格式
5. **质量损失**：压缩会导致一定的质量损失，建议根据场景调整质量参数
