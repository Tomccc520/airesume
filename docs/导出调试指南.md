## 🔍 手动测试步骤

### 步骤1：检查元素是否存在

在Firefox控制台（F12）中输入：

```javascript
const element = document.getElementById('resume-preview')
console.log('✅ 元素存在:', !!element)
console.log('📏 元素尺寸:', element?.getBoundingClientRect())
console.log('📝 内容长度:', element?.innerHTML.length)
console.log('👶 子元素数量:', element?.children.length)
```

**预期结果**：
- 元素存在: true
- 内容长度: 大于0的数字（例如：12345）
- 子元素数量: 1或更多

### 步骤2：手动测试导出功能

如果元素存在，在控制台输入以下代码手动测试导出：

```javascript
async function testExport() {
  const element = document.getElementById('resume-preview')
  
  if (!element) {
    console.error('❌ 找不到元素')
    return
  }
  
  console.log('✅ 找到元素')
  console.log('📏 尺寸:', element.getBoundingClientRect())
  console.log('📝 内容长度:', element.innerHTML.length)
  
  // 动态导入html2canvas
  const html2canvas = (await import('https://cdn.jsdelivr.net/npm/html2canvas@1.4.1/+esm')).default
  
  console.log('🎨 开始生成canvas...')
  
  const canvas = await html2canvas(element, {
    scale: 2,
    useCORS: true,
    allowTaint: true,
    backgroundColor: '#ffffff',
    logging: true
  })
  
  console.log('✅ Canvas生成完成:', { width: canvas.width, height: canvas.height })
  
  // 下载PNG
  const link = document.createElement('a')
  link.download = 'test-export.png'
  link.href = canvas.toDataURL('image/png')
  link.click()
  
  console.log('✅ 导出完成')
}

// 运行测试
testExport()
```

## 🐛 常见问题排查

### 问题1：元素不存在
**症状**：`element` 为 `null`

**解决方案**：
1. 确认你在 `/editor` 页面（不是首页）
2. 检查页面是否完全加载
3. 查看是否有React渲染错误

### 问题2：内容长度为0
**症状**：`innerHTML.length` 为 0

**解决方案**：
1. 检查是否添加了简历内容
2. 查看模板是否正确渲染
3. 检查CSS是否隐藏了内容

### 问题3：Canvas尺寸为0
**症状**：`canvas.width` 或 `canvas.height` 为 0

**解决方案**：
1. 检查元素的 `getBoundingClientRect()` 是否有尺寸
2. 确认元素不是 `display: none`
3. 检查是否有CSS transform影响

### 问题4：导出按钮不响应
**症状**：点击按钮没有反应

**解决方案**：
1. 检查控制台是否有JavaScript错误
2. 确认React组件是否正确挂载
3. 尝试硬刷新（Ctrl+Shift+R）

## 🔧 临时解决方案

如果导出功能完全不工作，可以使用浏览器的打印功能：

1. 按 **Ctrl+P**（Windows/Linux）或 **Cmd+P**（Mac）
2. 选择"另存为PDF"
3. 调整页面设置：
   - 纸张大小：A4
   - 边距：无或最小
   - 背景图形：开启
4. 保存PDF
