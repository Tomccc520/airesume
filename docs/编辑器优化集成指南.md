# 编辑器优化集成指南

## 概述

本文档说明如何将新创建的优化组件集成到编辑器页面中。

## 新增组件

### 1. EmptyState - 空状态组件
**文件**: `src/components/EmptyState.tsx`

**功能**: 当简历为空时显示引导界面

**集成方法**:
```tsx
import EmptyState from '@/components/EmptyState'

// 在 EditorPage 组件中添加状态
const [showEmptyState, setShowEmptyState] = useState(true)

// 检查简历是否为空
const isResumeEmpty = () => {
  return !resumeData.personalInfo.name && 
         resumeData.experience.length === 0 &&
         resumeData.education.length === 0 &&
         resumeData.skills.length === 0
}

// 在渲染中使用
{isResumeEmpty() && showEmptyState ? (
  <EmptyState
    onStartFromScratch={() => {
      setShowEmptyState(false)
      // 聚焦到第一个输入框
    }}
    onUseTemplate={() => {
      // 打开模板选择器
      setShowTemplateSelector(true)
    }}
    onAIGenerate={() => {
      handleAIGenerateResume()
      setShowEmptyState(false)
    }}
    onImportData={() => {
      // 触发文件选择
      const input = document.createElement('input')
      input.type = 'file'
      input.accept = '.json'
      input.onchange = (e) => {
        const file = (e.target as HTMLInputElement).files?.[0]
        if (file) {
          const reader = new FileReader()
          reader.onload = (e) => {
            try {
              const data = JSON.parse(e.target?.result as string)
              setResumeData(data)
              setShowEmptyState(false)
            } catch (error) {
              alert('导入失败，请检查文件格式')
            }
          }
          reader.readAsText(file)
        }
      }
      input.click()
    }}
  />
) : (
  // 原有的编辑器内容
  <ResumeEditor ... />
)}
```

### 2. ExportPreviewDialog - 导出预览对话框
**文件**: `src/components/ExportPreviewDialog.tsx`

**功能**: 导出前预览和选择格式

**集成方法**:
```tsx
import ExportPreviewDialog from '@/components/ExportPreviewDialog'

// 添加状态
const [showExportDialog, setShowExportDialog] = useState(false)

// 修改导出处理函数
const handleExportClick = () => {
  setShowExportDialog(true)
}

// 添加 JSON 导出功能
const handleExportJSON = () => {
  const dataStr = JSON.stringify(resumeData, null, 2)
  const blob = new Blob([dataStr], { type: 'application/json' })
  const url = URL.createObjectURL(blob)
  const link = document.createElement('a')
  link.href = url
  link.download = `${resumeData.personalInfo.name || '简历'}.json`
  link.click()
  URL.revokeObjectURL(url)
}

// 在渲染中使用
<ExportPreviewDialog
  isOpen={showExportDialog}
  onClose={() => setShowExportDialog(false)}
  onExport={(format) => {
    if (format === 'json') {
      handleExportJSON()
    } else {
      handleExport(format)
    }
  }}
  resumeName={resumeData.personalInfo.name}
/>
```

### 4. useVersionHistory - 版本历史 Hook
**文件**: `src/hooks/useVersionHistory.ts`

**功能**: 版本管理、撤销/重做

**集成方法**:
```tsx
import { useVersionHistory } from '@/hooks/useVersionHistory'

// 在 EditorPage 组件中使用
const {
  versions,
  currentVersionIndex,
  createVersion,
  restoreVersion,
  undo,
  redo,
  canUndo,
  canRedo
} = useVersionHistory(resumeData, {
  maxVersions: 10,
  autoSaveInterval: 60000 // 每分钟自动保存一个版本
})

// 添加撤销/重做按钮到工具栏
<button
  onClick={() => {
    const previousData = undo()
    if (previousData) {
      setResumeData(previousData)
    }
  }}
  disabled={!canUndo}
  className="..."
>
  撤销
</button>

<button
  onClick={() => {
    const nextData = redo()
    if (nextData) {
      setResumeData(nextData)
    }
  }}
  disabled={!canRedo}
  className="..."
>
  重做
</button>

// 手动创建版本
<button
  onClick={() => createVersion('手动保存')}
  className="..."
>
  保存版本
</button>
```

### 5. VersionHistoryPanel - 版本历史面板
**文件**: `src/components/VersionHistoryPanel.tsx`

**功能**: 显示和管理版本历史

**集成方法**:
```tsx
import VersionHistoryPanel from '@/components/VersionHistoryPanel'

// 添加状态
const [showVersionHistory, setShowVersionHistory] = useState(false)

// 在渲染中使用
<VersionHistoryPanel
  isOpen={showVersionHistory}
  onClose={() => setShowVersionHistory(false)}
  versions={versions}
  currentVersionIndex={currentVersionIndex}
  onRestore={(versionId) => {
    const versionData = restoreVersion(versionId)
    if (versionData) {
      setResumeData(versionData)
    }
  }}
  onDelete={(versionId) => {
    deleteVersion(versionId)
  }}
/>

// 添加打开按钮到工具栏
<button
  onClick={() => setShowVersionHistory(true)}
  className="..."
>
  <Clock className="w-4 h-4" />
  版本历史
</button>
```

## 性能优化

1. **懒加载对话框**
```tsx
const ExportPreviewDialog = dynamic(() => import('@/components/ExportPreviewDialog'), {
  ssr: false
})

const VersionHistoryPanel = dynamic(() => import('@/components/VersionHistoryPanel'), {
  ssr: false
})
```

2. **防抖处理**
```tsx
import { useMemo } from 'react'
import debounce from 'lodash/debounce'

const debouncedCreateVersion = useMemo(
  () => debounce(() => createVersion('自动保存'), 5000),
  [createVersion]
)

// 在数据变化时调用
useEffect(() => {
  if (!isResumeEmpty()) {
    debouncedCreateVersion()
  }
}, [resumeData, debouncedCreateVersion])
```
