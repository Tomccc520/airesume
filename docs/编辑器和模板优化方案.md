## 🎨 1. 编辑器界面优化（Editorial Modern 风格）

### 当前问题
- 使用通用蓝色配色
- 圆角过多（rounded-full）
- 缺乏独特性

### 优化方案

#### 1.1 样式设置面板优化
```typescript
// 配色更新
- 主色：#2563eb (蓝色) → #d97706 (琥珀金)
- 边框：1px → 2px 实线边框
- 圆角：rounded-full → rounded-lg (8px)
- 字体：Inter → Crimson Pro (标题) + DM Sans (正文)
```

#### 1.2 表单控件优化
- **输入框**：2px 边框，琥珀金聚焦状态
- **按钮**：深墨色背景 + 琥珀金文字
- **滑块**：琥珀金轨道和滑块
- **下拉菜单**：方形设计，Editorial 边框

#### 1.3 编辑器卡片优化
- 使用 2px 实线边框
- 悬停状态：琥珀金边框
- 标题使用 Crimson Pro 字体
- 内容使用 DM Sans 字体

## 📝 技术实现细节

### 1. 编辑器样式更新
```typescript
// 更新主题色
const EDITOR_THEME = {
  primary: '#d97706',      // 琥珀金
  secondary: '#1c1917',    // 深墨色
  accent: '#f59e0b',       // 浅琥珀
  border: '2px solid',     // 2px 边框
  borderRadius: '8px',     // 方形圆角
  fontDisplay: 'Crimson Pro',
  fontBody: 'DM Sans'
}
```

### 2. 模板数据结构
```typescript
interface TemplateStyle {
  id: string
  name: string
  nameZh: string
  description: string
  descriptionZh: string
  thumbnail: string
  colors: {
    primary: string
    secondary: string
    accent: string
    text: string
    background: string
  }
  fonts: {
    heading: string
    body: string
    mono: string
  }
  layout: {
    type: 'single-column' | 'two-column' | 'asymmetric'
    headerStyle: string
    sectionStyle: string
  }
  spacing: {
    section: number
    item: number
    padding: number
  }
}
```

### 3. 样式同步机制
```typescript
// 使用 useCallback 优化性能
const updateStyleSetting = useCallback((key: string, value: any) => {
  setIsPreviewUpdating(true)
  
  // 100ms 后隐藏更新状态
  setTimeout(() => {
    setIsPreviewUpdating(false)
  }, 100)
  
  // 更新样式配置
  updateStyleConfig({ [key]: value })
}, [updateStyleConfig])
```
