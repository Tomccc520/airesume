## 📖 详细使用说明

### 一、撤销/重做系统 🔄

#### 功能特点
- ⌨️ 支持键盘快捷键
- 💾 最多保存50个历史记录
- 🚀 性能优化，不影响编辑流畅度
- 🎯 智能检测状态变化

#### 使用方法

**键盘快捷键：**
- **撤销**: `Ctrl+Z` (Windows) / `Cmd+Z` (Mac)
- **重做**: `Ctrl+Shift+Z` (Windows) / `Cmd+Shift+Z` (Mac)
- **重做**: `Ctrl+Y` (Windows)

**代码集成：**
```typescript
import { useUndoRedo, useUndoRedoShortcuts } from '@/hooks/useUndoRedo'

function EditorPage() {
  // 使用撤销/重做
  const { 
    state: resumeData,
    setState: setResumeData,
    undo,
    redo,
    canUndo,
    canRedo,
    historySize,
    futureSize
  } = useUndoRedo(initialResumeData, {
    maxHistory: 50,
    onUndo: () => console.log('撤销操作'),
    onRedo: () => console.log('重做操作')
  })

  // 启用键盘快捷键
  useUndoRedoShortcuts(undo, redo, true)

  return (
    <div>
      <button onClick={undo} disabled={!canUndo}>
        撤销 ({historySize})
      </button>
      <button onClick={redo} disabled={!canRedo}>
        重做 ({futureSize})
      </button>
    </div>
  )
}
```

#### 配置选项
```typescript
interface UseUndoRedoOptions {
  maxHistory?: number      // 最大历史记录数，默认50
  enabled?: boolean        // 是否启用，默认true
  onUndo?: () => void     // 撤销回调
  onRedo?: () => void     // 重做回调
  onChange?: (state) => void  // 状态变更回调
}
```

### 二、版本管理系统 📚

#### 功能特点
- 🤖 自动版本保存（每小时）
- 📝 手动创建版本（带备注）
- 🔄 版本恢复
- 📊 版本对比
- 💾 版本导入导出
- 🏷️ 版本标签

#### 使用方法

**1. 保存版本**
```typescript
import { VersionManager } from '@/utils/versionManager'

// 手动保存版本
const version = VersionManager.saveVersion(
  resumeData,
  '完成工作经历部分',  // 备注
  false,                // 非自动保存
  ['milestone']         // 标签
)

// 自动保存版本
if (VersionManager.shouldAutoSave()) {
  VersionManager.saveVersion(resumeData, undefined, true)
}
```

**2. 获取版本列表**
```typescript
// 获取所有版本
const versions = VersionManager.getVersions()

// 获取版本统计
const stats = VersionManager.getStats()
console.log(`共有 ${stats.total} 个版本`)
console.log(`总大小: ${VersionManager.formatSize(stats.totalSize)}`)
```

**3. 恢复版本**
```typescript
// 恢复到指定版本
const restoredData = VersionManager.restoreVersion(versionId)
if (restoredData) {
  setResumeData(restoredData)
}
```

**4. 版本对比**
```typescript
// 对比两个版本
const diff = VersionManager.compareVersions(versionId1, versionId2)
console.log('修改的字段:', diff.modified)
```

**5. 导入导出**
```typescript
// 导出版本
VersionManager.exportVersion(versionId)

// 导入版本
const file = event.target.files[0]
const imported = await VersionManager.importVersion(file)
```

**6. 版本管理**
```typescript
// 删除指定版本
VersionManager.deleteVersion(versionId)

// 删除所有自动版本
const deletedCount = VersionManager.deleteAutoVersions()

// 清空所有版本
VersionManager.clearAllVersions()
```

#### 版本数据结构
```typescript
interface Version {
  id: string              // 版本ID
  timestamp: Date         // 创建时间
  data: ResumeData       // 简历数据
  note?: string          // 版本备注
  auto: boolean          // 是否自动创建
  tags?: string[]        // 版本标签
  size: number           // 文件大小（字节）
}
```

#### 实用工具函数

**工作经历 (30%)**
- 有工作经历 (30分)
- 经历数量 (20分)
- 经历质量：描述完整度、长度 (50分)

**教育背景 (20%)**
- 有教育背景 (40分)
- 教育数量 (20分)
- 教育质量：信息完整度 (40分)

**技能专长 (15%)**
- 有技能 (30分)
- 技能数量：建议5-10个 (30分)
- 技能多样性 (20分)
- 技能等级分布 (20分)

**项目经验 (15%)**
- 有项目经验 (30分)
- 项目数量 (20分)
- 项目质量：描述、技术栈、亮点 (50分)

## 🚀 集成到编辑器

### 完整示例

```typescript
'use client'

import { useState } from 'react'
import { useUndoRedo, useUndoRedoShortcuts } from '@/hooks/useUndoRedo'
import { VersionManager } from '@/utils/versionManager'
import ResumeScoreCard from '@/components/ResumeScoreCard'

export default function EditorPage() {
  // 1. 使用撤销/重做
  const {
    state: resumeData,
    setState: setResumeData,
    undo,
    redo,
    canUndo,
    canRedo
  } = useUndoRedo(initialData, {
    maxHistory: 50,
    onChange: (newData) => {
      // 自动保存版本
      if (VersionManager.shouldAutoSave()) {
        VersionManager.saveVersion(newData, undefined, true)
      }
    }
  })

  // 2. 启用快捷键
  useUndoRedoShortcuts(undo, redo)

  // 3. 手动保存版本
  const handleSaveVersion = () => {
    const note = prompt('请输入版本备注:')
    if (note) {
      VersionManager.saveVersion(resumeData, note, false, ['manual'])
      alert('版本保存成功！')
    }
  }

  return (
    <div className="editor-page">
      {/* 工具栏 */}
      <div className="toolbar">
        <button onClick={undo} disabled={!canUndo}>撤销</button>
        <button onClick={redo} disabled={!canRedo}>重做</button>
        <button onClick={handleSaveVersion}>保存版本</button>
      </div>

      {/* 评分卡片 */}
      <ResumeScoreCard 
        resumeData={resumeData}
        showDetails={true}
      />

      {/* 编辑器 */}
      <ResumeEditor
        resumeData={resumeData}
        onUpdate={setResumeData}
      />
    </div>
  )
}
```

## 🆘 常见问题

### Q1: 撤销/重做不生效？
A: 确保使用 `setState` 而不是直接修改 state，并检查是否启用了快捷键。

### Q2: 版本保存失败？
A: 检查 LocalStorage 空间是否充足，可以清理旧版本释放空间。

### Q3: 评分不准确？
A: 评分基于预设规则，可以根据实际需求调整 `resumeScorer.ts` 中的评分逻辑。

### Q4: 如何导出所有版本？
A: 遍历版本列表，逐个调用 `VersionManager.exportVersion()`。
